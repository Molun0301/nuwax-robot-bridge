# 文档导航

本目录存放适合公开阅读和上传到 GitHub 的介绍性文档。

如果你第一次接触这个项目，建议按下面顺序阅读：

1. [../../README.md](../../README.md)
   先了解项目定位、能力范围和仓库结构。
2. [../../部署文档.md](../../部署文档.md)
   按步骤完成宿主机部署、容器接入、MCP 注入和开机自启。
3. [平台架构说明.md](平台架构说明.md)
   了解平台分层、能力组织方式和扩展边界。
4. [宿主机与容器通信说明.md](宿主机与容器通信说明.md)
   了解宿主机与容器的职责分工和标准通信路径。

本目录只保留公开介绍类内容：

1. 项目是什么。
2. 项目如何部署。
3. 项目如何与容器通信。
4. 项目的能力结构与扩展方式。

实现细节、规划拆解、治理规范和内部研发资料不放在本目录中。
